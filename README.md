# 공지사항 관리 REST API

### 기능
1. 공지사항 등록, 수정, 삭제, 조회 API
2. 공지사항 등록시 입력 항목
   - 제목, 내용, 공지 시작일시, 공지 종료일시, 첨부파일 (여러개)
3. 공지사항 조회시 응답
   - 제목, 내용, 등록일시, 조회수, 작성자 

### 비기능 요구사항
1. RESTful한 API 설계
2. 기능 및 제약사항에 대한 단위/통합테스트 작성 
3. 대용량 트래픽을 고려하여 구현
